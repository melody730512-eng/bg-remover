# 📸 快速去背小工具 (Background Remover)

這是一個基於 Python `rembg` 與 `Gradio` 開發的網頁版去背工具。

### ✨ 功能特點
* **一鍵去背**：上傳圖片即可自動移除背景。
* **透明 PNG**：成品直接輸出為帶有透明通道的 PNG 格式。
* **網頁操作**：無需安裝軟體，開啟網頁即可使用。

### 🍰 適用場景
* **甜點創作**：製作肉桂捲或手工甜點的宣傳圖。
* **新聞編輯**：快速處理去背素材。
* **Vibe Coding**：讓瑣事自動化的數位副手。

### 🚀 如何運行
1. 安裝需求套件：`pip install -r requirements.txt`
2. 執行程式：`python app.py`
